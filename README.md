# GestionProductosAPI

API REST desarrollada con ASP.NET Core para la gestión de productos.

## Tecnologías utilizadas

- C#
- ASP.NET Core
- Entity Framework Core
- SQL Server
- AutoMapper
- FluentValidation
- Repository Pattern

## Funcionalidades

- Obtener todos los productos
- Obtener producto por ID
- Crear producto
- Actualizar producto
- Eliminar producto

## Arquitectura del proyecto

El proyecto sigue una estructura organizada:

- Controllers
- DTOs
- Services
- Repository Pattern
- Validaciones con FluentValidation
- Mapeo con AutoMapper
